## Legend

- **TEAL/BLOK** - Teal/green building blocks (various sizes)
- **PURP/ROOF** - Purple tiled roof pieces
- **ROOF** - Roof components (peaks, edges, corners)
- **WOOD/WALL** - Wood panel wall segments
- **WIN/DOW** - Window sections (white frames)
- **WALL/MIX** - Mixed material walls
- **WALL/CORN** - Corner wall pieces
- **FLOR/TILE** - Floor/foundation tiles
- **TABL/E** - Table furniture
- **STOL/E** - Stool/seating
- **FLAT** - Flat roof or platform tiles
- **GABL** - Gabled roof sections

## Extraction Command Examples

### Extract Single Tile
```bash
convert "assets/iso-64x64-building (1).png" \
  -crop 64x64+0+0 \
  +repage \
  "output/tile-0-0.png"
```

### Extract Row
```bash
convert "assets/iso-64x64-building (1).png" \
  -crop 640x64+0+0 \
  +repage \
  "output/row-0.png"
```

### Extract All Tiles (Loop)
```bash
for row in {0..7}; do
  for col in {0..9}; do
    x=$((col * 64))
    y=$((row * 64))
    convert "assets/iso-64x64-building (1).png" \
      -crop 64x64+${x}+${y} \
      +repage \
      "output/tile-${col}-${row}.png"
  done
done
```

## Recommended Groupings

### Group 1: Simple Buildings
- Foundation blocks + wood walls + flat teal roof
- Positions: (0,2), (2,2)-(4,2), (0,0)-(2,0)

### Group 2: Fancy Buildings
- Foundation + mixed walls + purple tiled roof
- Positions: (2,3), (6,3), (3,0)-(5,0), (1,1)-(5,1)

### Group 3: Houses with Windows
- Walls + window sections + gabled roof
- Positions: (0,3)-(5,3), (4,1), (5,1)

### Group 4: Interior Furnishing
- Floor tiles + furniture
- Positions: (0,6)-(4,6)

## Priority Tiles for Initial Extraction

**High Priority (Most Useful):**
1. (2,2) - Standard wood wall
2. (3,3) - Single window
3. (4,1) - Gabled roof left
4. (5,1) - Gabled roof right
5. (0,0) - Teal block foundation
6. (2,6) - Table
7. (9,3) - Wall corner

**Medium Priority:**
- All window variants (row 3-4)
- All roof pieces (rows 0-1)
- Floor tiles (row 6)

**Low Priority:**
- Large block variations
- Duplicate similar pieces
- Edge fragments
